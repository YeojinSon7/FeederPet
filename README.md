## 1. 프로젝트 설명
- 이 프로젝트는 3D 프린터를 통해 제품을 개발하고 아두이노를 연결하여 작동시키는 프로젝트입니다.
- 반려동물 자동급식기를 만들었으며 사용자가 원할 때마다 앱의 버튼을 눌러 사료가 나오게 할 수 있습니다.

## 2. 작동 영상
